Interview Questions

Prepare for your Accenture interview with these commonly asked questions across different categories:

What is the difference between abstract class and interface in Java?

An abstract class can have both abstract and non-abstract methods, while an interface can only have abstract methods (prior to Java 8). Abstract classes can have constructors, instance variables, and different access modifiers, whereas interfaces have only public static final variables and public abstract methods by default. A class can implement multiple interfaces but can extend only one abstract class.

Explain the concept of normalization in databases.

Normalization is the process of organizing data in a database to reduce redundancy and improve data integrity. It involves dividing large tables into smaller ones and defining relationships between them. The main forms of normalization are:

  • 1NF: Eliminate repeating groups and ensure atomic values
  • 2NF: Meet 1NF requirements and remove partial dependencies
  • 3NF: Meet 2NF requirements and remove transitive dependencies
  • BCNF: A stronger version of 3NF
What is the time complexity of quicksort in worst, average, and best cases?

The time complexity of quicksort is:

  • Worst case: O(n²) - occurs when the pivot is always the smallest or largest element
  • Average case: O(n log n)
  • Best case: O(n log n) - occurs when the pivot always divides the array into equal halves

If a train travels at 60 km/hr and crosses a platform in 30 seconds, what is the length of the platform if the train is 200 meters long?

To solve this problem:

  1. Convert the speed to meters per second: 60 km/hr = 16.67 m/s
  2. The total distance traveled = (train length + platform length)
  3. Time taken = 30 seconds
  4. Using the formula: Distance = Speed × Time
  5. Total distance = 16.67 × 30 = 500 meters
  6. Platform length = Total distance - Train length = 500 - 200 = 300 meters
If 6 men can complete a work in 12 days working 8 hours per day, how many days will it take for 8 men to complete the same work working 6 hours per day?

This is a work-time problem that can be solved using the formula:

(M₁ × D₁ × H₁) = (M₂ × D₂ × H₂)

Where M = number of men, D = number of days, H = hours per day

Given: M₁ = 6, D₁ = 12, H₁ = 8, M₂ = 8, H₂ = 6, and we need to find D₂

Substituting the values:

(6 × 12 × 8) = (8 × D₂ × 6)

576 = 48D₂

D₂ = 12 days

Find the next number in the series: 3, 7, 15, 31, 63, ___

To find the pattern, look at the differences between consecutive terms:

  • 7 - 3 = 4
  • 15 - 7 = 8
  • 31 - 15 = 16
  • 63 - 31 = 32

The differences are doubling each time: 4, 8, 16, 32. So the next difference would be 64.

Therefore, the next number in the series is 63 + 64 = 127.

Correct the sentence: "Neither of the students were able to solve the problem."

The correct sentence is: "Neither of the students was able to solve the problem."

Explanation: "Neither" is singular, so it takes a singular verb "was" instead of the plural "were."

Choose the word that best completes the sentence: "The project manager was _____ by the team's progress."

Possible answers:

  • Impressed
  • Impressing
  • Impressive
  • Impression

The correct answer is "Impressed." The sentence requires a past participle to complete the passive voice construction.

Read the passage and answer: What is the main idea of the given text?

This type of question tests your reading comprehension skills. To answer correctly:

  1. Read the entire passage carefully
  2. Identify the central theme or purpose
  3. Distinguish between main ideas and supporting details
  4. Look for repeated concepts or ideas that tie the passage together
  5. Consider the author's intention and the context

Practice with reading comprehension exercises to improve your skills in identifying main ideas quickly.
