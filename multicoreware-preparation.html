Reverse a singly linked list without using extra nodes.

Solution Approach:

To reverse a singly linked list in-place, we need to change the direction of all the pointers:

Node* reverseLinkedList(Node* head) {
    Node* prev = nullptr;
    Node* current = head;
    Node* next = nullptr;
    
    while (current != nullptr) {
        // Store next
        next = current->next;
        
        // Reverse current node's pointer
        current->next = prev;
        
        // Move pointers one position ahead
        prev = current;
        current = next;
    }
    
    // prev is the new head
    return prev;
}

Time Complexity: O(n) where n is the number of nodes in the linked list

Space Complexity: O(1) as we're only using a constant amount of extra space

Explain the difference between malloc and calloc.

Both malloc and calloc are used for dynamic memory allocation in C, but they differ in several ways:

  • Initialization: malloc doesn't initialize the allocated memory, leaving it with garbage values. calloc initializes all bits to zero.
  • Parameters: malloc takes one parameter (size in bytes), while calloc takes two parameters (number of elements and size of each element).
  • Return value: Both return a void pointer to the allocated memory, or NULL if allocation fails.
  • Usage: malloc is typically faster when you plan to initialize the memory yourself. calloc is preferred when you need zero-initialized memory.

Example:

// Allocating memory for 5 integers using malloc
int* arr1 = (int*)malloc(5 * sizeof(int));

// Allocating memory for 5 integers using calloc
int* arr2 = (int*)calloc(5, sizeof(int));
Implement a dynamic 2D array using pointer arithmetic.

There are multiple ways to implement a dynamic 2D array in C. Here's an implementation using a single contiguous block of memory:

int** create2DArray(int rows, int cols) {
    // Allocate memory for row pointers
    int** arr = (int**)malloc(rows * sizeof(int*));
    if (arr == NULL) {
        return NULL; // Memory allocation failed
    }
    
    // Allocate memory for the entire 2D array in one block
    int* data = (int*)malloc(rows * cols * sizeof(int));
    if (data == NULL) {
        free(arr);
        return NULL; // Memory allocation failed
    }
    
    // Set up row pointers
    for (int i = 0; i < rows; i++) {
        arr[i] = data + (i * cols);
    }
    
    return arr;
}

// Function to free the 2D array
void free2DArray(int** arr) {
    // Free the data block
    free(arr[0]);
    // Free the row pointers
    free(arr);
}

This implementation allows you to access elements using the familiar arr[i][j] syntax while maintaining memory efficiency.
